Welcome to Image Similarity API

This API allows you to upload an image and find similar images based on cosine similarity.

Endpoints:

  • Upload Image: POST /upload

    Upload an image to find similar images.

    Request:

    • file: The image file to be uploaded.

    Response: A JSON array with similar images and their similarity scores.

Example Request:

POST http://localhost:5000/upload

Upload an image file using a tool like Postman or via a form on your frontend.
